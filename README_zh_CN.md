## 插件介绍

这个插件是为思源笔记的主题[VSCode Lite Edit](https://github.com/emptylight370/siyuan-vscode-lite)开发的，仅在安装有此主题时生效。

目前插件本身不实现任何单独的功能，全部功能需要依靠主题运行。并且不会将主题的主要功能移动到插件中。
